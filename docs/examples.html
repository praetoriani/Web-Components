Web Components > Examples  
Example: Badge-Widget
Know those fancy badges from shields.io?
Here you can get it as a web component.

You can set the following attributes:
The background color as well as a shadow and a glossy effect. You got two text options and you can even put an icon on the badge.
Example: Simple Popup
This example shows you a simple popup window as a ready-made web component.

You can set the following attributes:
The size and the the title of the window as well as the text shown on the popup and the text for the button.
Example:Tooltip Popup
Wanna get my tooltip popups I'm using here? Okay. Get them as a re-usable web component.

Following attributes can be set:
The title, body text and font/fontsize as well as size and position of the tooltip. And you can set the level for the z-axis
(to show the tooltip on top of all elements).
Made with HTML 🧩 , CSS 🖌 and Javascript 🧰

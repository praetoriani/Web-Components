Hello This is the
Modal Window Web Component

The Modal Window can be configured in different ways.
You can choose if you want a title or not.
You can set one, two or three buttons on the window.
You can choose between a dark and light theme.
You can use a 'shield'
(the shield blocks the interaction with the rest of the website)

And of course you can set the width and height
as well as the position (top/left) of the window Cool! Wow! Did I tell you that you can define individual functions for each button?

You can assign a click-event for each button. And that means you can assign your own functions to the buttons. Pretty cool, huh?!
